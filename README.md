# To-Do List Web (Frontend)

Este repositório contém apenas o **frontend** da aplicação To-Do List.

## Tecnologias

- React
- React Router v6
- Axios para requisições HTTP
- CSS / styled-components

## Funcionalidades

- Login e registro de usuários
- Visualização e gerenciamento de tarefas
- Navegação entre páginas (SPA)
- Comunicação com o backend via API

## Site Online

A aplicação frontend está disponível em:  
[https://to-do-list-web-chi.vercel.app/](https://to-do-list-web-chi.vercel.app/)
